Aplikasi ini digunakan untuk memantau dan mengelola downtime mesin, serta simulasi gaji.

Menu ini digunakan untuk mencatat data downtime mesin, meliputi NIK, factory, area, ID mesin, masalah, tindakan, sparepart, serta waktu mulai dan selesai.

  • Tabel: Monitoring data downtime berdasarkan Nama Mekanik, Tanggal, Factory, dan Area.
  • Grafik: Analisa data downtime secara visual berdasarkan Tanggal.
  • Summary: Monitoring input data downtime per Mekanik.
  • Top ID Mesin: Menampilkan ID mesin yang paling sering rusak.

Menu ini digunakan untuk menghitung Gaji Bruto (belum termasuk potongan) berdasarkan gaji pokok, tunjangan, lembur, dan insentif.

  • Gunakan sidebar di sebelah kiri untuk berpindah antar menu.
  • Pastikan input Waktu Mulai dan Waktu Selesai benar.
  • Jika ada kendala penggunaan aplikasi, silakan hubungi bagian TPM.
